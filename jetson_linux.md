# Jetson Linux Setup

1. Register as a Nvidia Developer at https://developer.nvidia.com/login (This step is required to use the Nvidia SDK Manager)

2. Install the Nvidia SDK Manager, read https://www.jetson-ai-lab.com/tutorials/initial-setup-sdk-manager/

3. Place a screwdrivver head under the pin 9 and pin 10 of the J14 header to short the pins, in order to set the kit in Forced Recovery mode.

4. Connect the kit to a Linux host via USB-C port, and turn on the power.

5. Follow the steps at https://www.jetson-ai-lab.com/tutorials/initial-setup-sdk-manager/ to complet the Jetson Linux installation.

6. Remove the screwdriver.
